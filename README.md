# ConnectSphere

ConnectSphere is a Django-based social media platform where users can create profiles, share posts, follow other users, like and comment on posts, and connect with the community.

## Features

- User Authentication
- Create & Edit Profile
- Create, Update & Delete Posts
- Like & Comment System
- Follow / Unfollow Users
- Search Users
- Responsive UI
- Password Reset

## Tech Stack

- Python
- Django
- HTML
- CSS
- Bootstrap
- JavaScript
- SQLite3
